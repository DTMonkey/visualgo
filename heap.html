The ACTIONS panel on the bottom left displays all available operations on this data structure. Experiment with them!

The STATUS panel on the right displays the status of the current operation. The CODETRACE panel below it displays the pseudocode for the current operation, and uses code highlighting to show you which step is being carried out.

During each action, you can use the playback controls at the bottom of the screen to control the animation. Note that you have to click 'stop' first before you can issue another action.
